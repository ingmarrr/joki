# Let Statement

Simple assignment of a value to a variable.

```jo
let x: u16 = 5;
let y: f32 = 1.5;
let z: str = "Hello World";
let c: char = 'c';
let b: bool = true;

// Todo: Add support for type inference
let a := 5;
```
